# Credit Card Fraud Detection

## Objective
Identify fraudulent transactions using anomaly detection and machine learning models.

## Tools Used
Python, Pandas, Scikit-learn, XGBoost, Matplotlib, Seaborn

## Workflow
- Data preprocessing and cleaning
- Handling missing values
- Anomaly detection using Isolation Forest and LOF
- Supervised learning using XGBoost
- Model evaluation using confusion matrix and ROC curve

## Results
- Achieved high accuracy in detecting fraudulent transactions
- ROC curve shows strong model performance
- Confusion matrix highlights prediction results

## Conclusion
The project demonstrates the effectiveness of combining anomaly detection and classification techniques for fraud detection.
